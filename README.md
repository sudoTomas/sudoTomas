### 🔭 Featured project

**[embry0](https://github.com/sudoTomas/embry0)** — an AI-native software delivery engine. It takes a task from a plain-language issue to a reviewed pull request by orchestrating a pipeline of specialized agents (triage → development → review → end-to-end QA) as a LangGraph state graph, with human oversight at every gate and full auditability. Untrusted agent code runs in hardened Docker-in-Docker sandboxes with credential-injection proxies, so it never touches real secrets or the host. Full-stack: a React 19 control plane over a FastAPI + PostgreSQL backend.
